# 🤖 چت‌بات هوشمند کاملاً سمت‌کلاینت

یک چت‌بات هوشمند پیشرفته که به طور کامل در مرورگر اجرا می‌شود. بدون نیاز به سرور، بدون API خارجی و با حفظ حریم خصوصی کامل.

![License](https://img.shields.io/badge/License-MIT-blue)
![WebGPU](https://img.shields.io/badge/WebGPU-Required-orange)

## ✨ ویژگی‌های کلیدی

- **💯 کاملاً سمت کلاینت**: هیچ داده‌ای به سرور ارسال نمی‌شود
- **🧠 هوش مصنوعی لوکال**: اجرای مدل WebLLM در مرورگر
- **🔍 سیستم پاسخ‌دهی هوشمند**: ترکیب FAQ + هوش مصنوعی
- **🎨 طراحی مدرن**: تم روشن/تیره، واکنش‌گرا
- **📱 ویجت آماده**: نصب آسان در هر سایتی
- **💾 ذخیره‌سازی محلی**: تاریخچه گفتگو در LocalStorage
